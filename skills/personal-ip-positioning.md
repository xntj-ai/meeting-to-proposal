# personal-ip-positioning · 个人 IP 定位

> 基于「张拼拼·XNTJ」EP0013 公开 demo 的实际产出反推。
> 把客户访谈中的「成长经历 / 职业生涯 / 个人特质」转成可执行的 IP 定位。

## 何时调用

被 [`meeting-to-proposal`](../SKILL.md) orchestrator 在 Step 5 自动调用；也可以独立使用 —— 用户给出某人的背景资料，要求做个人 IP 定位时。

## 输入

| 字段 | 必要 | 说明 |
|---|---|---|
| 成长经历 | ✓ | 教育 / 早期工作 / 关键转折点 |
| 职业生涯 | ✓ | 主要工作经历 / 行业积累 |
| 个人特点特质 | ✓ | 性格 / 思维方式 / 表达风格 |
| 当下业务 | ✓ | 现在的公司 / 服务方向 / 客户类型 |
| 目标受众 | 可选 | 想吸引谁来加你 |

## 核心方法 · 5 个分析维度

### 1. 经历提炼 · 找「跨度」与「深度」

从受访者的成长 + 职业生涯里，找出两类信号：

- **跨度**：跨过几个行业 / 国家 / 角色（越多越能讲反共识视角）
- **深度**：在某个领域沉淀的年限（越长越能讲专业）

输出格式：

```
跨度：跨过 N 个行业、M 个国家、X 个角色
深度：在 Y 领域 Z 年专业积累
```

### 2. 特质拆解 · 找「IP 暗号」

从访谈中提取 3-5 个高频出现的**个人特质标签**。比如：

- 表达层面：直白 / 反共识 / 故事化 / 数据化
- 思维层面：跨界 / 第一性原理 / 反思型 / 行动派
- 工作风格：一人公司 / 团队主理 / 高频日更 / 深度长文

每个特质给一个**具体场景例子**作证（不能凭印象编造）。

### 3. 差异化优势 · 找「跨界三角」

把上面的「跨度」「深度」「特质」交叉组合，找出**只有他能讲**的差异化角度：

```
只有他能讲的事情 = 深度领域 × 跨界经历 × 个人特质
```

EP0013 demo 实例参考：「新加坡公司创始人 + 跨境出海专业积累 + 想做个人 IP」→ 差异化角度是「为中国企业家在新加坡建立确定性的安全感」（地缘 + 专业 + 情绪锚点）。

### 4. 价值主张 · 一句话讲清楚

格式：

```
我帮 [谁]，通过 [方法]，实现 [结果]。
```

例：
- 「我帮**想出海新加坡的中国企业家**，通过**本地化的商业洞察 + 全流程协助**，实现**降低试错成本 + 建立确定性的安全感**。」

### 5. 内容人设 · 5 个长期标签

定 5 个**长期不变的标签**（一年内别换）。原则：

- 包含 1 个**身份标签**（角色 / 行业）
- 包含 1 个**反差标签**（反共识或反预期的点）
- 包含 1 个**情绪标签**（让人愿意 follow 的情感联结）
- 包含 1 个**专业标签**（让人愿意付费的专业证明）
- 包含 1 个**生活标签**（让人觉得是个真人的细节）

EP0013 demo 参考张拼拼自己：「🐧2 进 2 出 / 17 年跨 7 个行业 / 一人公司副业 8 年 / 40 岁 All in AI / XNTJ.AI 拥抱智序创始人」

## 输出格式

最终交付一份 Markdown 或 HTML section，4 个区块：

1. **经历提炼**（跨度 + 深度）
2. **特质标签**（3-5 个，每个带场景例子）
3. **差异化优势**（一段话讲清楚跨界三角）
4. **价值主张 + 5 个长期标签**

## 调用提示词

```
请你利用 personal-ip-positioning skill，基于访谈中的成长经历、职业生涯、个人特质，
做完整的个人 IP 定位分析。按 5 个维度输出。
```

## 关键约束

1. **不凭印象编造特质** — 每个特质必须有访谈中的场景例子作证
2. **不抄竞品定位** — 跨界三角必须是这个人独有的组合
3. **5 个长期标签别频繁换** — 一年内别动，让受众形成认知锚定
